# Checklists CTQ (Critical to Quality) — binaires

Chaque item = OUI ou NON. Si NON : citer le passage problématique et corriger,
ou documenter dans la section Limites.

## CTQ universels (tous documents)

| # | Critère |
|---|---------|
| U1 | Chaque affirmation factuelle a une source traçable (URL, référence, article) |
| U2 | Le document répond au cas d'usage défini en phase 0 |
| U3 | Aucune section vide ou remplie de vague |
| U4 | Terminologie cohérente (pas de synonymes flottants) |
| U5 | Le langage est adapté au destinataire final identifié |
| U6 | Les interdictions de la spec sont respectées |
| U7 | Pas de contradiction interne entre sections |
| U8 | Les arbitrages entre sources contradictoires sont explicités et justifiés |

## CTQ supplémentaires — Document SST

| # | Critère |
|---|---------|
| S1 | Les articles du Code du travail cités sont en vigueur (post-réforme 2021 si applicable) |
| S2 | Les VLEP citées sont à jour (vérifiées via SSTinfo) |
| S3 | Les tableaux de MP référencés existent et correspondent |
| S4 | La distinction obligation de moyens / résultats est correcte |
| S5 | Les préconisations relèvent du champ de compétence et de la mission du MdT (prévention, aptitude, non-thérapeutique) |
| S6 | **SECRET MÉDICAL** : aucune donnée de santé dans les livrables destinés à l'employeur ou au CSE |
| S7 | Anonymisation appliquée dès la rédaction ([SALARIÉ], [ENTREPRISE], etc.) |

## CTQ supplémentaires — Document juridique / opposable

| # | Critère |
|---|---------|
| J1 | Chaque article de loi cité est vérifié via un appel SSTinfo (Légifrance) |
| J2 | La jurisprudence citée est sourcée (n° pourvoi, date, juridiction) |
| J3 | Les objections adverses prévisibles sont anticipées |
| J4 | Le document résiste à une lecture par la partie adverse |
| J5 | Toute affirmation juridique sans base textuelle vérifiée (loi, décret, JP) est supprimée |

## CTQ supplémentaires — Document clinique / médical

| # | Critère |
|---|---------|
| M1 | Les recommandations citées sont les plus récentes disponibles |
| M2 | Les niveaux de preuve sont indiqués quand pertinent |
| M3 | Les effets indésirables / contre-indications sont mentionnés |
| M4 | La chronologie clinique est cohérente et vérifiable |
| M5 | Le langage et les concepts sont adaptés au destinataire (expert, juge, patient, employeur) |

## Règle d'usage

En phase 0, sélectionner les CTQ applicables (universels + domaine).
En phases 1 et 4, cocher chaque item OUI ou NON.
Si NON → citer le passage et corriger, ou justifier dans Limites.
Tous les items critiques (S6, S7, J5) = OUI obligatoire pour livrer, sans exception.
